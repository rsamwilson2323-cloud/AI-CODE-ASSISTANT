# 🤖 AI CODE ASSISTANT

AI CODE ASSISTANT is an intelligent developer tool that helps programmers understand, convert, analyze, and test code using AI-powered utilities.  
The application provides multiple coding tools inside a single interface to improve productivity and learning.

It is built using **React + Vite** and provides a clean and interactive UI for developers and students.

---

# 🚀 Features

### 🧠 Code Explainer
Understand any piece of code instantly.  
Paste your code and the AI assistant will generate a simplified explanation.

### 🔄 Code Converter
Convert code between different programming languages.

Example:
- Python → Java
- Java → C++
- C++ → JavaScript

### ⚡ Code Compiler
Compile and execute code directly from the interface.

### ⏱ Time Complexity Analyzer
Analyze algorithm time complexity such as:
- O(1)
- O(log n)
- O(n)
- O(n²)

### 💾 Space Complexity Analyzer
Understand how much memory your algorithm uses.

### 🎨 Modern UI
Simple and responsive interface designed for smooth interaction.

# ⚙️ Installation

### 1️⃣ Clone the repository

```
git clone https://github.com/rsamwilson2323-cloud/AI-CODE-ASSISTANT.git
```

### 2️⃣ Move to the project folder

```
cd AI-CODE-ASSISTANT
```

### 3️⃣ Install dependencies

```
npm install
```

This installs all required packages listed in **package.json**.

---

# ▶️ Running the Project

Start the development server:

```
npm run dev
```

Vite will start the server.

Open browser and go to:

```
http://localhost:5173
```

# 🖥 Application Workflow

### Step 1 — Open the Application
The homepage loads and displays different AI coding tools.

### Step 2 — Choose a Tool
Available tools include:

- Code Explainer
- Code Converter
- Code Compiler
- Time Complexity Analyzer
- Space Complexity Analyzer

### Step 3 — Input Code
Paste your program or algorithm into the input box.

### Step 4 — Process the Code
The application processes the code using the selected tool.

### Step 5 — View Result
The output will display:

- Explanation
- Converted code
- Complexity analysis
- Compilation result
